## Introduction

In this project, you'll create an Excel (.xlsm) file with macros that can calculate macronutrient needs for users based on their input. Whether you're a nutrition enthusiast, a fitness professional, or a developer interested in Excel macros, this project allows you to create a useful tool for calculating macronutrient requirements.

## Features

- **User Inputs:** Allow users to input information such as weight, activity level, and goals.
- **Macronutrient Calculation:** Implement formulas and macros to calculate protein, carbohydrates, and fat requirements.
- **Result Display:** Display the calculated macronutrient needs in an organized format.

## Getting Started

To get started with this project, follow the instructions below:

### Prerequisites

Before you begin, ensure you have the following:

- Microsoft Excel installed on your computer.
- Basic understanding of Excel functions and macros (VBA).

### Installation

1. Clone this repository to your local machine using the following command:

   ```bash
   git clone https://github.com/ameer-ali-jamali/Assignment-7-Macro-Calculator-MS-Excel.git
   ```

2. Open the Excel file provided in this project (`.xlsm` extension).

## Usage

1. Open the Excel file provided in this project (`.xlsm` extension).
2. Enable macros if prompted (make sure the file is from a trusted source).
3. Input the necessary user information (weight, activity level, goals, etc.).
4. Run the macro to calculate macronutrient needs.
5. View the calculated macronutrient requirements on the spreadsheet.

Feel free to customize and enhance the calculator to include additional features or improve the user interface.
